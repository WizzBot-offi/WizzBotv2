# 🤖 WizzBot V2

WizzBot V2 is an upgraded hardware + web system that combines IoT with a custom-built web interface for real-time interaction and control.
It connects physical components with a digital interface, allowing control and monitoring through a browser.
---

## 🚀 Features
- Real-time device control using NodeMCU (ESP8266)
- Integration with Arduino for hardware operations
- Cloud communication using Adafruit IO
- Custom-built web interface for interaction
- Live data monitoring and control system
---

## 🛠 Tech Stack
- Arduino
- NodeMCU (ESP8266)
- Adafruit IO
- HTML, CSS, JavaScript
---

## ⚙️ How It Works
WizzBot V2 connects hardware components to a web-based interface:
1. NodeMCU handles WiFi communication  
2. Arduino manages hardware logic  
3. Adafruit IO acts as the cloud bridge  
4. The website provides a user interface for control  
This creates a system where users can interact with hardware in real-time from the browser.
